# 📊 Ecommerce Dashboard - Power BI

## 📌 Overview
This project presents an interactive Ecommerce Dashboard built using Power BI to analyze sales performance, profit trends, customer behavior, and payment preferences. The dashboard helps in understanding business insights through dynamic filters and visualizations.
## 🎯 Features
- Sales, Purchase Cost & Profit KPIs
- Product-wise Sales & Profit Analysis
- Region-wise Filter
- Payment Method Analysis
- Gender & Category Filters
- Interactive Visuals

## 🛠 Tools Used
- Power BI
- Data Visualization
- Data Cleaning & Transformation
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset
